## 📌 Project Overview

This project is an interactive Retail Sales Dashboard developed in Microsoft Excel to analyze sales performance, profitability, customers, stores, and product categories.

The dashboard is built using Microsoft Excel's advanced analytical features and follows a modern retail dashboard design inspired by Zudio's branding.

---

## 🚀 Key Features

- Interactive Dashboard
- Dynamic KPI Cards
- Sales Trend Analysis
- Profit Trend Analysis
- State-wise Sales
- Clothing Type Analysis
- Category Analysis
- Parking Availability Analysis
- Store Type Analysis
- Interactive Slicers

---

## 📈 KPIs

- Total Sales
- Total Orders
- Total Profit
- Average Order Value
- Total Stores
- Total Staff
- Total Customers
